== Description ==

Adds Modal Login, Register, Lost Password & Facebook Login functionality

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `sq-login-booster.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `sq-login-booster.zip`
2. Extract the `sq-login-booster` directory to your computer
3. Upload the `sq-login-booster` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
